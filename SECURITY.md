# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in torguard-lite, **please open a GitHub issue** at:

https://github.com/wildfirebill-web/torguard-lite/issues/new

Do not email — all security concerns are handled through the public issue tracker.

Please include:
- A description of the issue
- Steps to reproduce
- Affected version
- Severity estimate (low / medium / high / critical)
